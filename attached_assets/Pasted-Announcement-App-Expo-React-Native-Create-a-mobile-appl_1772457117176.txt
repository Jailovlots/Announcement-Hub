Announcement App (Expo + React Native)

Create a mobile application called **“ZDSPGC Announcement App”** using **React Native with Expo**.

The app is a school announcement system with two roles: **Admin and Student**.

Use:

* Expo (latest version)
* React Navigation
* Firebase (Authentication + Firestore)
* Expo Notifications
* Functional components with React Hooks
* Clean folder structure


---

## 👥 USER ROLES

Implement role-based authentication:

* If role === "admin" → Navigate to *Admin Dashboard*
* If role === "student" → Navigate to *Student Dashboard*

---

## 🔐 AUTHENTICATION SYSTEM

Create:

1. Login Screen

   * Email
   * Password
   * Login button

2. Register Screen (Students only)

   * Name
   * Email
   * Password

3. Store user role in Firestore:
   Collection: users

   
   {
     id: string,
     name: string,
     email: string,
     role: "admin" | "student",
     createdAt: timestamp
   }
   

4. Admin account is pre-created manually in Firebase.

5. Add logout functionality.

## 👨‍🎓 STUDENT FEATURES

### Student Dashboard

* Display in FlatList

* Search bar

  * Academic
  * Events
  * Emergency
  * General

### Announcement Details Screen

* Title
* Description
* Category
* Date Posted
* Posted By
* Optional Image

Firestore Collection: announcements

{
  id: string,
  title: string,
  description: string,
  category: string,
  imageUrl: string,
  createdAt: timestamp,
  createdBy: string
}
